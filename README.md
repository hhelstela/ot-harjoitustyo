## TaskApp
TaskApp on on kalenteri, johon kirjataan pääsääntöisesti tapahtumien sijaan tehtäviä, jotka pitäisi sen päivän aikana tehdä. Sovelluksen on tarkoitus auttaa käyttäjää ajanhallinnassa ja aikaansaamisessa.
